# The Second Serve Project

A small Costa Rican project against everyday waste. Colorful bins for used tennis balls, designed to live on country clubs, academies, and schools — and a quiet argument that small, well-designed things stack.

Made by [Gianluca Fonseca](https://secondserve.today/founder), 17, civil engineer.

- **Live:** [secondserve.today](https://secondserve.today)
- **Instagram:** [@secondserve_cr](https://instagram.com/secondserve_cr)

## Stack

- Next.js 15 (App Router) · React 19 · TypeScript
- Tailwind CSS v4
- EB Garamond + Inter via `next/font`

## Routes

- `/` — landing page
- `/mission` — why this exists, and the engineering mindset behind it
- `/founder` — Gianluca's bio
- `/poster/[color]` — printable Spanish posters for each bin (`red`, `yellow`, `blue`, `green`)

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
